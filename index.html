Developer Instructions

To add new movies permanently, edit the hardcodedMovies array in the JavaScript code:

// Add your movies here (this will be saved in GitHub) const hardcodedMovies = [ { id: 1, title: "Movie Title", category: "Action", iframeCode: '<iframe src="YOUR_VIDEO_URL"></iframe>', description: "Movie description" }, // Add more movies here... ];

After adding movies, upload this HTML file to GitHub Pages. The movies will be permanently visible to all visitors.
